This file contains the actual prompts I used to build and improve features in my project using AI tools like ChatGPT as well as Claude AI.

## 1. Dark Mode Toggle (JavaScript)

**Prompt:**
I am building a simple website using HTML, CSS, and JavaScript.  
I want to add a dark mode toggle button in the navbar.  

When the user clicks the button:
- The website should switch between light and dark mode  
- A class like "dark" should be added/removed from the body  
- The toggle button icon should change  

Please provide clean and beginner-friendly JavaScript code.

**Output:**
- Dark mode toggle implemented  
- Dynamic icon switching  
- Clean and simple logic using `classList.toggle()`
